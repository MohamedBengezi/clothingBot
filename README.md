# clothingBot


## Requirements
- python 3.6
- pip3
- Firefox webdriver: https://github.com/mozilla/geckodriver/releases


## Installation
- Run `pip install -r requirements.txt` to install required packages


## Usage
- Run `python .\clothingBot.py "URL"`
